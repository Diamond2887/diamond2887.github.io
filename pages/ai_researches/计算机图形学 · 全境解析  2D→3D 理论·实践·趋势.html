一、计算机图形学总论与概述
1.1 学科定义与研究范畴

计算机图形学（Computer Graphics，简称CG） 是一门研究如何利用计算机技术生成、处理和显示图形图像的学科，其本质是使用数学算法将二维或三维形体转化为计算机图形的科学。作为生成虚拟世界的基础技术，计算机图形学能够将真实世界的对象物体在相应的三维虚拟世界中重构，并保存其部分物理属性，通过逼真的计算机图形构筑出以假乱真的虚拟世界，使使用者产生身临其境的感觉。

从技术构成来看，计算机图形学的基本元素主要包括四个核心方面：建模（表示选择、几何处理）、渲染（几何变换、可见性、光的模拟）、交互（输入输出设备、工具）和动画（逼真角色、自然现象及其交互）。这四个元素相互关联，共同构成了计算机图形学的完整技术体系。

跨学科特征：线性代数是计算机图形学最重要的数学工具，包括矩阵运算、向量空间、特征值等；光学原理为光照模型和渲染算法提供理论支撑，材料科学为材质属性的真实感模拟奠定了基础。

1.2 发展历程与技术演进

六个重要阶段：1950年代开创（被动式图形，MIT旋风I号）；1960年代快速发展（Sketchpad，Sutherland，Bézier曲线）；1970年代光栅化时代（光反射模型，Phong光照模型）；1980年代真实感渲染（光线跟踪，辐射度）；1990-2000硬件加速（GPU诞生，可编程着色器）；2000至今可编程管线及AI驱动（Vulkan/DX12，RTX，DLSS）。

1962年Ivan E. Sutherland发表“Sketchpad：一个人机交互通信的图形系统”，首次使用Computer Graphics术语。1975年Phong提出著名光照模型。1999年NVIDIA发布GeForce 256，首次提出GPU概念。2018年实时光线追踪（RTX）开启混合渲染新时代。

1.3 技术发展现状与趋势

2025年处于AI驱动的智能化转型期，视觉大模型参数达数千亿，多模态处理。三大支柱：光线追踪硬件化（RT Core）、PBR成为标准渲染范式、AI神经网络渲染（NVIDIA RTX神经网络渲染技术，DLSS 4，最高8倍性能提升）。

SIGGRAPH会议是世界上规模最大的专业会议，1997年参会达48700人。图形学标准化（ANSI、ISO，CGI、CGM、GKS、PHIGS）推动产业发展。从特效市场到数字孪生，图形学成为数字经济的核心驱动。

二、2D图形学核心技术
2.1 基础图形元素绘制算法

Bresenham直线算法 (1965) — 整数加法与误差修正，避免浮点运算。主轴选择策略，整数型误差项e。

中点Bresenham画圆：八分法对称，整数判别式。椭圆及其他二次曲线同样采用整数增量绘图框架。

2.2 光栅化原理与多边形填充

光栅化：判断像素是否在图形内。三角形填充常用叉乘法（符号一致）。扫描线算法：利用奇偶性确定填充区间，支持复杂多边形。现代GPU并行化光栅化，深度测试、模板混合。

2.3 坐标变换与几何操作

仿射变换 = 线性变换 + 平移。缩放、旋转、平移，齐次坐标统一为3×3矩阵。旋转矩阵正交，缩放分为一致缩放/差值缩放。齐次坐标系统用三维向量表示2D点。

2.4 裁剪算法与优化策略

Sutherland-Hodgman逐边裁剪法：窗口每条边裁剪多边形，四种位置关系。通用凸凹多边形，任意凸窗口。Cohen-Sutherland编码裁剪：快速判断完全可见/不可见。现代GPU硬件裁剪、视锥体裁剪。

2.5 反走样技术与图像质量

反走样：SSAA、MSAA、FXAA、TAA。边缘检测+颜色混合，通过低通滤波平滑锯齿。现代GPU支持硬件MSAA、CSAA、EQAA。AI抗弯技术崭露头角。

三、3D图形学核心技术
3.1 MVP矩阵变换体系

MVP = Model (模型空间→世界) × View (世界→相机) × Projection (相机→裁剪)。模型矩阵负责位置/旋转/缩放；视图矩阵由相机位置、目标、上方向定义；投影矩阵分为透视投影（近大远小）与正交投影。组合顺序 P×V×M，齐次坐标4×4矩阵。

3.2 着色器技术架构

着色器 (GLSL) ：顶点着色器（坐标变换，法线，gl_Position）；片段着色器（像素颜色，纹理采样）；几何着色器（生成图元、粒子、LOD）；GLSL精度限定符 (highp/mediump)。可编程管线颠覆固定功能。

3.3 现代渲染管线架构

几何处理 → 光栅化 → 片段处理。可编程着色器、曲面细分、几何着色器。光线追踪硬件RT Core，混合渲染。MRT多渲染目标，延迟渲染。

3.4 光照模型与材质系统

Phong模型：环境+漫反射+镜面反射；Blinn-Phong使用半角向量简化。PBR：基于微表面模型、能量守恒、物理BRDF（GGX法线分布，菲涅尔Schlick近似）。金属度/粗糙度工作流。

3.5 GPU架构与并行计算

1999 GeForce 256 (5700万晶体管) → 2025 Blackwell (920亿晶体管，21000+着色器)。统一着色器架构，张量核心(Tensor Core)，RT Core。HBM2e显存(48GB)。SIMT，并行处理。AI加速器驱动DLSS。
