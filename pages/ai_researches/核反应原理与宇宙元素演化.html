1. 宇宙起源与原初核合成

1.1 宇宙大爆炸与早期宇宙

宇宙的故事始于约138亿年前的一个极度炽热、致密的奇点。根据最新的宇宙学观测数据，这个由阿塔卡马宇宙学望远镜（ACT）于2025年3月发布的结果显示，宇宙年龄已被精确确定为138亿年，精度达到0.1%。

宇宙大爆炸并非我们通常理解的那种在空间中发生的爆炸，而是空间本身的急剧膨胀。在爆炸发生后的10^-43秒内，宇宙经历了被称为"普朗克时代"的极早期阶段，这个时间尺度小到我们目前的物理理论还无法完全描述。

随着宇宙的急速膨胀，它开始迅速冷却。在宇宙年龄达到1秒时，温度已经下降到约100亿度。这时，一个关键的转变发生了：原本在高温下自由存在的夸克和胶子开始结合，形成了质子和中子等基本粒子。

1.2 原初核合成过程

原初核合成（Primordial Nucleosynthesis）是宇宙演化史上的第一个"元素工厂"，发生在宇宙大爆炸后约1秒到20分钟的时间窗口内。这个过程决定了今天宇宙中90%以上物质的基本组成。

1.3 轻元素的形成与丰度

宇宙中元素的丰度分布呈现出明显的层次性特征。根据最新的观测数据，按质量计算，今天银河系中氢元素占74%，氦元素占24%，两者合计占宇宙总质量的98%。

锂元素的情况则更为特殊。理论预测的锂-7丰度比观测值高出3-4倍，这就是著名的"宇宙锂问题"。科学家们提出了多种解释，包括早期宇宙中可能存在的额外物理过程、恒星演化对锂的破坏，或者是我们对原初核合成过程的理解还不够完善。

2. 恒星演化与核反应循环

2.1 主序星阶段：氢聚变

当宇宙年龄达到约5亿年时，在引力的作用下，由氢原子和氦原子组成的气体云开始聚集收缩。当气体云核心的温度达到1000万度时，氢原子核之间的剧烈碰撞终于克服了它们之间的静电斥力，氢核聚变反应开始了。

质子-质子链反应（适用于太阳及质量更小的恒星）

1. 两个质子碰撞，其中一个通过弱相互作用转化为中子，形成氘核
2. 氘核与另一个质子结合，形成氦-3核
3. 两个氦-3核碰撞，形成氦-4核并释放两个质子

碳氮氧循环（CNO循环，适用于质量大于1.1倍太阳质量的恒星）

CNO循环是一个催化过程，碳、氮、氧原子核在反应中起到催化剂的作用。

2.2 红巨星阶段：氦聚变与三重α过程

当恒星核心的氢燃料耗尽后，恒星进入了生命的晚期阶段。当红巨星核心温度达到约1亿度时，新的核反应开始了——氦聚变。

三重α过程包含两个步骤：

1. 两个氦-4核（α粒子）结合形成铍-8核
2. 铍-8与第三个氦-4核碰撞，形成稳定的碳-12核

一旦碳-12形成，它还可以继续与氦-4核反应生成氧-16、氖-20等元素。宇宙中90%的碳和约三分之二的氧都是通过这种方式产生的。

2.4 超新星爆发与重元素诞生

超新星爆发是宇宙中最剧烈的天体物理现象之一，也是比铁更重元素的主要诞生地。

r过程（快中子俘获过程）

在超新星爆发的极端环境中，中子密度达到10^20-10^22 cm^-3，温度高达10^9 K。在这种条件下，原子核可以快速俘获大量中子，形成从铁到铀的所有重元素。

s过程（慢中子俘获过程）

s过程发生在红巨星的氦燃烧阶段，中子密度较低（约10^8-10^10 cm^-3），中子俘获速度较慢。这个过程缓慢而稳定，主要产生原子序数在铅、铋附近的元素。

3. 核反应的基本原理

3.1 原子核结构与核力

原子核由质子和中子组成，它们统称为核子。原子核的基本性质包括：

  • 原子序数（Z）：质子数，决定了元素的化学性质
  • 质量数（A）：质子数+中子数，近似等于原子的质量
  • 中子数（N）：A-Z，决定了同位素的特性

核力的特性

核力是自然界四种基本相互作用之一，具有以下特点：

  • 短程力：核力的作用范围极短，只有约10^-15米
  • 强相互作用：核力比电磁力强100倍，是自然界最强的力
  • 饱和性：每个核子只与邻近的核子发生作用
  • 电荷无关性：核力对质子和中子的作用相同

3.2 结合能与质量亏损

结合能是理解核反应能量来源的关键概念。结合能是指将原子核中的所有核子完全分开所需要的能量，或者反过来，是核子结合成原子核时释放的能量。

根据爱因斯坦的质能方程E=mc²，质量和能量是等价的。当核子结合成原子核时，会发生质量亏损：

Δm = 核子总质量 - 原子核质量

这个质量亏损Δm通过E=Δmc²转化为结合能释放出来。

比结合能曲线揭示了核反应的能量规律：

  • 聚变反应：轻核聚合成中等质量核时，比结合能增加，释放能量
  • 裂变反应：重核分裂成中等质量核时，比结合能增加，释放能量
  • 铁核的特殊性：铁-56的比结合能最高，无论是聚变还是裂变都需要吸收能量

3.3 核裂变机制

核裂变是指重原子核分裂成两个或多个中等质量原子核的过程。这个过程最早在1938年由德国科学家奥托·哈恩和弗里茨·斯特拉斯曼发现。

裂变的基本过程

1. 重核（如铀-235）吸收一个中子，形成激发态的复核
2. 复核发生形变，从球形变为椭球形
3. 当形变超过临界值时，核力无法维持原子核的整体，原子核分裂成两个碎片
4. 同时释放2-3个中子和大量能量

以铀-235为例，典型的裂变反应为：

²³⁵U + n → ¹⁴¹Ba + ⁹²Kr + 3n + 200MeV

链式反应

裂变反应的一个重要特征是能够形成链式反应，这是原子弹和核反应堆的基础。

3.4 核聚变机制

核聚变是指轻原子核结合成一个较重原子核的过程。这是太阳和其他恒星的能量来源，也是未来清洁能源的希望。

聚变的基本条件

要实现核聚变，必须克服两个原子核之间的库仑势垒：

  • 需要极高的温度（10^8 K以上）使原子核具有足够的动能
  • 需要足够高的密度和约束时间，使聚变反应能够持续进行

主要的聚变反应

1. 氘-氚反应（D-T）：D + T → ⁴He + n + 17.6MeV
2. 氘-氘反应（D-D）：D + D → ³He + n + 3.27MeV
3. 氘-氦-3反应（D-³He）：D + ³He → ⁴He + p + 18.3MeV

最新进展

2025年，核聚变研究取得了重大突破：

  • 中国"东方超环"（EAST）实现1亿摄氏度1066秒稳态等离子体运行
  • 中国环流三号实现原子核温度1.17亿度、电子温度1.6亿度的参数水平
  • 美国国家点火装置（NIF）在2025年4月实现8.6兆焦耳聚变产额，能量增益系数Q值达4.13

4. 同位素与核稳定性

4.1 同位素概念与分类

同位素是指具有相同质子数但中子数不同的原子。它们在元素周期表中占据同一位置，化学性质几乎相同，但物理性质存在差异。

同位素的表示方法

同位素通常用符号表示，如：

氢的同位素：
¹H（氢）、²H（氘）、³H（氚）
碳的同位素：
¹²C、¹³C、¹⁴C
铀的同位素：
²³⁵U、²³⁸U

同位素的分类

根据稳定性，同位素可分为：

  • 稳定同位素：不会自发衰变的同位素
  • 放射性同位素：会自发衰变的同位素，具有一定的半衰期

4.2 核稳定性规律

原子核的稳定性遵循一定的规律，这些规律对理解核反应和同位素应用至关重要。

决定核稳定性的因素

  • 中子-质子比（N/Z）：随着Z增加，需要更多的中子来抵消质子间的电磁斥力
  • 幻数效应：当质子数或中子数为特定数值（2, 8, 20, 28, 50, 82, 126）时，原子核特别稳定
  • 偶奇性：偶偶核（Z和N均为偶数）最稳定，占稳定核素的58%

4.4 超重元素探索

超重元素是指原子序数大于103的元素，它们的合成和研究是核物理和化学的前沿领域。

超重元素的合成方法

  • 冷熔合反应：用重离子轰击锕系元素靶，主要用于合成104-112号元素
  • 热熔合反应：用⁴⁸Ca轰击重锕系元素靶，成功合成了113-118号元素

理论预测在Z=114、N=184附近存在一个"稳定岛"，岛上的超重元素可能具有相对较长的半衰期。
