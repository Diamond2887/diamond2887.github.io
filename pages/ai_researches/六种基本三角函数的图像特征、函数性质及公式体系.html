三角函数的几何意义

三角函数是数学中重要的基本函数，它们源于对三角形边角关系的研究，后推广到任意角的情形。在直角三角形中，设角θ是一个锐角，斜边长为c，对边长为a，邻边长为b，则六个基本三角函数的定义如下：

  • 正弦函数：sin(θ) = 对边/斜边 = a/c
  • 余弦函数：cos(θ) = 邻边/斜边 = b/c
  • 正切函数：tan(θ) = 对边/邻边 = a/b
  • 正割函数：sec(θ) = 斜边/邻边 = c/b = 1/cos(θ)
  • 余割函数：csc(θ) = 斜边/对边 = c/a = 1/sin(θ)
  • 余切函数：cot(θ) = 邻边/对边 = b/a = 1/tan(θ)

在单位圆（半径为1的圆）中，设角θ的顶点在原点，始边沿x轴正方向，终边与单位圆交于点P(x, y)，则三角函数可以表示为点P的坐标分量：

  • sin(θ) = y
  • cos(θ) = x
  • tan(θ) = y/x
  • sec(θ) = 1/x
  • csc(θ) = 1/y
  • cot(θ) = x/y
六种三角函数的基本关系

六种三角函数之间存在着密切的关系，这些关系构成了三角函数运算的基础。

倒数关系

  • sin(θ) × csc(θ) = 1
  • cos(θ) × sec(θ) = 1
  • tan(θ) × cot(θ) = 1

商数关系

  • tan(θ) = sin(θ)/cos(θ) = sec(θ)/csc(θ)
  • cot(θ) = cos(θ)/sin(θ) = csc(θ)/sec(θ)

平方关系

  • sin²(θ) + cos²(θ) = 1
  • 1 + tan²(θ) = sec²(θ)
  • 1 + cot²(θ) = csc²(θ)

这些基本关系为三角函数的化简、求值和证明提供了重要工具。

周期性

三角函数最显著的性质之一是周期性，这源于它们在单位圆上的定义。周期性意味着函数值在一定间隔后重复出现。

基本周期规律

  • 正弦函数和余弦函数的周期为2π
  • 正切函数和余切函数的周期为π
  • 正割函数和余割函数的周期为2π

周期性的数学表达为：

  • sin(x + 2π) = sin(x)
  • cos(x + 2π) = cos(x)
  • tan(x + π) = tan(x)
  • cot(x + π) = cot(x)
  • sec(x + 2π) = sec(x)
  • csc(x + 2π) = csc(x)

两角和与差的三角函数公式

两角和与差的三角函数公式是三角恒等式的重要组成部分，它们允许我们将两个角的和或差的三角函数表示为单个角三角函数的组合。

cos(α + β) = cos(α)cos(β) - sin(α)sin(β)
tan(α + β) = [tan(α) + tan(β)] / [1 - tan(α)tan(β)]

tan(α - β) = [tan(α) - tan(β)] / [1 + tan(α)tan(β)]
